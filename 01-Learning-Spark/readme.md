## 🛠 핵심 학습 키워드

### 1. 스파크 기초 및 구조
- **개념 이해:** 트랜스포메이션(Transformation), 액션(Action), 지연 평가(Lazy Evaluation)
- **Spark UI:** 애플리케이션 실행 모니터링 및 성능 확인
- **정형화 API:** RDD, 데이터 프레임(DataFrame), 데이터세트(Dataset)의 차이와 활용

### 2. 데이터 소스 및 SQL
- **내장/외부 연동:** JSON, Parquet, CSV, JDBC(PostgreSQL) 연결
- **고차 함수:** 스파크 SQL을 활용한 데이터 가공 및 분석

### 3. 성능 최적화
- **튜닝:** 데이터 캐싱 및 영속화(Persistence) 전략
- **조인 전략:** Shuffle Hash Join, Broadcast Hash Join 등 상황별 조인 방식
- **최적화 엔진:** 카탈리스트(Catalyst) 옵티마이저 및 텅스텐(Tungsten) 프로젝트

### 4. 고급 기능
- **Structured Streaming:** 마이크로 배치 모델 및 상태 유지 집계
- **Data Lake:** 델타 레이크(Delta Lake)를 활용한 레이크하우스 구축
- **MLlib:** 머신러닝 파이프라인 설계 및 하이퍼파라미터 튜닝
